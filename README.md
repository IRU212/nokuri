# Nokuri

## 概要
筋トレを行うユーザ向けのサイトで天気によっておすすめの筋トレメニューを教えたり <br />
筋トレ用のWikiで様々な知識を蓄える事が出来ます。

## サーバー情報
ユーザ画面URL : http://www.nokuri.jp/  <br />
ドキュメントルート : ~/www/nokuri/src/public

管理画面URL : http://admin.nokuri.jp/  <br />
ドキュメントルート : ~/www/nokuri/src/admin
